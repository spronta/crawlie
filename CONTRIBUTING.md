We love and appreciate contributions from the community and thank you for taking the time to do so ❤️

**Here's how you can contribute:**
- [Open an issue](https://github.com/spronta/crawlie/issues) if you've come across a bug 
- [Create a pull request](https://github.com/spronta/crawlie/pulls) to add new features, make improvements or fix bugs


**If you like the project**, but just don't have time to contribute, that's totally cool. There are other easy ways to support the project and show your appreciation, which we would also be super happy about:
- Star the project here on GitHub ⭐
- Share it on social (X, Reddit or LinkedIn)
- Refer this project in your project's readme
- Mention the project at local meetups and tell your friends/colleagues
